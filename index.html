Northeastern's Course Schedule is Bad

Currently, to view your class schedule you must navigatte there from the Student Hub which is inconvient. Furthermore, the schedule is hard to read and provides no easy form of integration with any calendar or orgizational tool.

How MySchedule+ Helps

iCalendar (.ics) is a univerisal calendar file format that can be imported to a prexisitng calendar of your choice. Copy and paste your entire schedule into the box below to genorate your own .ics file.

How to Import an iCalendar File:

  • Outlook Calendar
  • Google calendar
  • Apple calendar

Note: Google Calendar will need the first day to be manually edited after import.

Using MySchedule+

Go to Student Hub > Course Registration and select the deesired term and then copy it with one of the methods below. Then paste your raw schedule in the input feild below and click genorate to download your iCalendar file.

  • Manualy select the page
  • Ctrl + A on Windows
  • Cmd + A on Mac
